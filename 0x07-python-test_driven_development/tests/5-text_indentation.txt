

================================
5-text_indentation module
================================

This module defines a text-indentation function ``text_indentation(text)``.

Usage
------

    >>> text_indentation = __import__('5-text_indentation').text_indentation
    >>> text_indentation("Best?")
    Best?
    <BLANKLINE>

No spaces are printed at the beginning of a line.



    >>> text_indentation("   Hi there.")
    Hi there.
    <BLANKLINE>



    >>> text_indentation("          ")

    >>> text_indentation("Best.   ")
    Best.
    <BLANKLINE>



    >>> text_indentation("    Best.    nice nicey.   ")
    Best.
    <BLANKLINE>
    nice nicey.
    <BLANKLINE>

    >>> text_indentation("there is no stop, just text")
    there is no stop, just text

New lines within a string are printed as normal.



    >>> text_indentation("Let's print a new-line! yes here:\nok Printed.")
    Let's print a new-line! yes here:
    <BLANKLINE>
    <BLANKLINE>
    ok Printed.
    <BLANKLINE>

    >>> text_indentation("\n\n\n 3 lines printed.")
    <BLANKLINE>
    <BLANKLINE>
    <BLANKLINE>
    3 lines printed.
    <BLANKLINE>


    >>> text_indentation("A brand-new \n new line.")
    A brand-new 
    new line.
    <BLANKLINE>
